# 1. Introduction to Object-Oriented Programming

**Object-Oriented Programming (OOP)** is a programming paradigm centered around
objects rather than actions. It promotes the use of classes and objects to model
real-world entities, enabling more organized, modular, and reusable code. The
core principles of OOP include:

- Encapsulation: Bundling data (attributes) and methods (functions) that operate
  on that data within objects.
- Inheritance: Creating new classes (subclasses) that inherit properties and
  behaviors from existing classes (superclasses).
- Polymorphism: Allowing objects of different classes to be treated as objects
  of a common superclass, often enabling the same method to behave differently
  based on the object.
- Abstraction: Simplifying complex reality by modeling classes appropriate to
  the problem, exposing only necessary parts.

JavaScript, while primarily a prototype-based language, supports OOP principles
through constructor functions and ES6 classes. Understanding these concepts is
crucial for building scalable and maintainable applications.
